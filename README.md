# @juicyarts/config-stylelint

Centralized linting rules for scss/css using stylelint.

## preqrequisits

- [stylelint](https://stylelint.io/) needs to be installd in your project

## use

create or edit a .stylelintrc.json

```json
{
  // ...
  "extends": [
    "@juicyarts/config-stylelint"
  ]
  // ...
}
```
